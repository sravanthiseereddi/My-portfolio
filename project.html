Palindrome finder

I created a palindrome finder using HTML, CSS, and JavaScript, where users can input text to check if it reads the same forward and backward. The HTML structures the input and output, CSS styles the interface, and JavaScript handles the palindrome logic.

Random number guessing game

I built a random number guessing game using HTML, CSS, and JavaScript, allowing users to guess a number within a set range. The HTML provides the layout, CSS styles the game interface, and JavaScript generates the random number and checks the user's guess.

To-do list

I developed a to-do list application using HTML, CSS, and JavaScript, where users can add, delete, and mark tasks as complete. HTML structures the list, CSS enhances the visual layout, and JavaScript manages the task functionality and interactivity.

Flower-themed website

I designed a flower-themed website using HTML, CSS, Bootstrap, and JavaScript to showcase different flower collections. HTML and Bootstrap structured the responsive layout, CSS added custom styling, and JavaScript enabled interactive features like image sliders and dynamic content.

Places website

I designed a places website using only HTML and CSS to display various travel destinations with images and brief descriptions. HTML structures the content, while CSS enhances the visual layout with styling, grid layouts, and responsive design.

Facts about numbers

I created a project to know facts about a number using HTML, CSS, and JavaScript, where users can enter any number to get fun or interesting facts. HTML provides the structure, CSS styles the interface, and JavaScript fetches number facts from an API and displays them dynamically.

Student application form

I developed a student application form using HTML, CSS, and JavaScript to collect and validate student details like name, age, course, and contact information. HTML structures the form, CSS styles it for a clean layout, and JavaScript adds interactive validation to ensure the data is entered correctly before submission.
